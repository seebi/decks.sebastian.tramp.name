Approach

Based on the given motivational points, the following contributions exists:

  • An architecture of a Distributed Semantic Social Network
  • Implementations of the architectures protocols
  • Implementation of different application prototypes, which verify and demonstrate this architecture

Architecture: Basic Principles I

  • Linked Data
    • main protocol for data publishing, retrieval and integration
    • de-referencable and interlinked resources
    • fosters extensibility and a distributive overall character
  • Service Decoupling
  • Protocol Minimalism

Architecture: Basic Principles II

  • Linked Data
  • Service Decoupling
    • unravel services / application from the users data
    • settles ownership to the user
    • fosters privacy and pushes distribution
  • Protocol Minimalism

Architecture: Basic Principles III

  • Linked Data
  • Service Decoupling
  • Protocol Minimalism
    • communicate RDF triples between DSSN nodes
    • do not enforce workflow or exact interpretation
    • ensures the extensibility of the data model
